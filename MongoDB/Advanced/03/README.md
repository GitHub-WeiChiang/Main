03 - 資料存取
=====
* ### MongoDB 在新增資料時，若資料庫與資料表不存在，會自動建立。
* ### 新增文件時，"_id" 欄位可以自行指定，若未指定或指定為 null 則 MongoDB 會使用 ObjectId() 函數生成。
* ### "_id" 欄位的型態可以是字典，但不能是陣列。
* ### 新增文件時，若 "_id" 欄位型態不正確或是該值已經存在皆會引發錯誤。
* ### 常用數字型態比較運算子
    * ### "$gt": 大於。
    * ### "$gte": 大於等於。
    * ### "$lt": 小於。
    * ### "$lte": 小於等於。
    * ### "$ne": 不等於。
